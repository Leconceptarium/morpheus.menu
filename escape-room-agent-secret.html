Escape Room – Extension : Agent Secret

Une mission d’infiltration sous haute tension !
60 min
3 à 5 joueurs
16 ans et +

Cette extension du jeu Escape Room vous plonge dans une mission d’espionnage où chaque seconde compte. Déjouez les pièges, décodez les messages secrets et infiltrez le repaire ennemi avant qu’il ne soit trop tard.

Nécessite le jeu de base Escape Room – Le Jeu pour être jouée.
